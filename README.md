# ProyectoFinalCiberseguridad4GeeksAccademy
Proyecto final de ciberseguridad de 4Geeks Academy: análisis forense de un servidor Debian comprometido, corrección del hackeo inicial, explotación y mitigación de una nueva vulnerabilidad en WordPress y diseño de un plan de respuesta a incidentes y SGSI basado en NIST 800-61 e ISO 27001.
